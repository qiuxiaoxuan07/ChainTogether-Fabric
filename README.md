# ChainTogether Fabric

ChainTogether is a Minecraft Fabric mod for 1.21.1 that lets players bind together with leads, render a visible chain line, and keep partners synchronized during movement, teleporting, and respawn flows.

## Commands

`/chain bind <player1> <player2>` - bind two players.

`/chain unbind <player>` - remove a player's binding.

`/chain length <multiplier>` - set the chain length multiplier.

`/chain reload` - reload the JSON config.

`/chain list` - list active player bindings.

`/chain transparency <10~100>` - set the rendered line transparency.
